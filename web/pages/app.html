About

 

Immersion App

A desktop application for spaced repetition flashcard study, immersion time tracking, and review session management.

Version 1.2.0

 

Features

  • Spaced repetition system (SRS) with customizable scheduling
  • Anki deck import (.apkg)
  • Custom card types and field templates
  • Review statistics and activity heatmap
  • Immersion time tracking with categories and stats
 

Built With
