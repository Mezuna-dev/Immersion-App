Features

Everything you need to learn a language

Immersion Suite gives you professional-grade SRS tools in a clean, fast desktop app. It's designed to be completely free, no account required.

🧠
Spaced Repetition (SRS)
SM2-based scheduling keeps reviews efficient. Study cards exactly when you need to, not before, not after.
  • Configurable learning & relearning steps
  • New first, mixed, or new last study order
  • Keyboard shortcuts during review
  • Audio autoplay for listening cards
📦
Anki Import
Import your existing Anki decks including card templates, media, and full review history.
  • Supports old and new .apkg formats
  • Imports note types & CSS styles
  • Images, audio, and zstd-compressed media
  • Full review history preserved
📊
Statistics & Dashboard
Visualize your progress with a heatmap, retention charts, and per-deck breakdowns. Know exactly where you stand.
  • Review activity heatmap
  • Retention stats by day / week / month / year
  • Due, new, young, and mature card counts
🗂️
Deck & Card Management
Create custom card types with your own fields and HTML/CSS templates or just import your Anki setup.
  • Multiple decks with per-deck settings
  • Custom fields and HTML/CSS templates
  • JSON backup and restore
  • Local-only, your data stays on your machine
🔌
Browser Extension
Hold Shift and hover over any Japanese text in Chrome or Firefox to instantly look it up.
  • Works on any webpage
  • Powered by Jitendex (~295k dictionary entries)
  • Shadow DOM popup with no CSS conflicts
  • 512-entry LRU cache for instant repeat lookups
🔒
Private by Design
No accounts, no subscriptions, no telemetry. All your data lives in a local SQLite database you own and control.
  • Fully offline
  • Open-source, inspect every line of code
  • Export your data any time as JSON
Browser Extension

Look up words without leaving the page

The Immersion Suite extension adds a hover dictionary to any webpage. Hold Shift, hover over Japanese text, and a popup appears with the reading, part of speech, and definition.

1
Install the desktop app

The extension is a thin UI. The desktop app does the heavy lifting (dictionary lookup, SRS).

2
Load the extension in Chrome or Firefox

Load the extension/ folder as an unpacked extension from the GitHub releases.

3
Hold Shift and hover

The extension talks to the desktop over a local WebSocket. No data leaves your machine.
