Features

Every Immersion Tool You Need

No subscriptions, no cloud, no nonsense. Just the tools you actually need, running on your machine.

🧠
Spaced Repetition (SRS)
Reviews show up when you're about to forget them, not randomly. SM2 scheduling handles the timing so you don't have to think about it.
  • Configurable learning & relearning steps
  • New first, mixed, or new last study order
  • Keyboard shortcuts during review
  • Audio autoplay for listening cards
📦
Anki Import
Already using Anki? Bring your decks over. Templates, media, and full review history all come with.
  • Supports old and new .apkg formats
  • Imports note types & CSS styles
  • Images, audio, and zstd-compressed media
  • Full review history preserved
📊
Statistics & Dashboard
See your review history, retention rates, and streaks at a glance. Hard to slack off when the heatmap's staring at you.
  • Review activity heatmap
  • Retention stats by day / week / month / year
  • Due, new, young, and mature card counts
⏱️
Immersion Tracking
Timer, manual logs, and a full media library. Track the anime you're watching, the books you're reading, and how many hours you've actually put in.
  • Built-in timer with category tagging
  • Manual log entry for past sessions
  • Pie chart breakdown by category
  • Media library with watching status
  • Anime & manga search via MyAnimeList
🗂️
Deck & Card Management
Build card types from scratch with custom fields and HTML/CSS templates, or just import your existing Anki setup and go.
  • Multiple decks with per-deck settings
  • Custom fields and HTML/CSS templates
  • JSON backup and restore
  • Local-only, your data stays on your machine
🔌
Browser Extension
Hold Shift and hover over any Japanese text in Chrome or Firefox to instantly look it up.
  • Works on any webpage
  • Powered by Jitendex (~295k dictionary entries)
  • Shadow DOM popup with no CSS conflicts
  • 512-entry LRU cache for instant repeat lookups
🔒
Private by Design
No accounts, no cloud sync, no telemetry. Your data is a SQLite file on your own machine. That's it.
  • Fully offline
  • Open-source, inspect every line of code
  • Export your data any time as JSON

2

Load the extension/ folder as an unpacked extension from the GitHub releases.

3
Hold Shift and hover

Everything stays local. The extension and app communicate over a WebSocket on your own machine.
